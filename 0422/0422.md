## 기본 데이터 타입 (Primitive data type)
- python에서 기본적으로 미리 정의해놓은 데이터 타입
- 숫자
    - 정수 (integer)
        - positive, negative 다 가능
        - 예) 3, 2341235, -23432, 0, 33
    - 실수 (float)
        - positive, negative 다 가능
        - 예) 3.134, -3.29348, 0.000, 2/3
- 문자 (string)
    - letter, word, sentence 이런거 구분 없음
    - string을 정의(사용)할 때 내용물을 ***따옴표 안에 넣어야 함!***
        - 작은 따옴표, 큰 따옴표 모두 가능
        - 예) "a", 'b', "wadu", "a b c", "33", "#"
    - string은 capital sensitive 함: "a" 랑 "A" 는 다른 문자
- 참/거짓 (Boolean, 혹은 줄여서 Bool)
    - 두 가지만 가능!
        - 참: True 
        - 거짓: False
    - 비슷한 string 주의!
        - True vs "True"
            - True는 bool
            - "True"는 string
        - False vs "False"
            - False는 bool
            - "False"는 string
- `type()` 함수
    - 내가 어떤 데이터를 줬어. 근데 걔가 무슨 타입인지 궁금해. 이 때 쓰는 함수가 `type()`
    - 예) 
        ```python
        >>> type("wadu")
        str
        ```

## 기본 데이터 타입의 기본 연산
### 숫자 (integer, float)

- 사칙연산
    - `+`: 더하기
    - `-`: 빼기
    - `*`: 곱하기
    - `/`: 나누기
        
        - 예)
            ```python
            >>> 9 / 9
            1.0
            ```

- 거듭제곱
    - `** `
        
        - a ** x == $`a^x`$ <pre xml:lang="latex">\sqrt{2}</pre> (a의 x승, a to the power of x)
        - 예)
            ```python
            >>> 3 ** 2
            9
            ```
- 몫과 나머지
    - `//`: 몫
        - a // b == a를 b로 나눈 몫
    - `%`: 나머지
        - a % b == a를 b로 나눈 나머지
- 비교
    - 같냐 다르냐
        - `==`
        
          - 같으냐
        
          - 예)
        
            ```python
            >>> 3 == 3.0
            True
            ```
        
        - `!=` 
        
          - 다르냐
        
          - 예)
        
            ```python
            >>> 1 != -1
            True
            ```
    - 대소비교
        - `>`
          - 크냐
        - `<`
          - 작냐
        - `>=`
          - 크거나 같냐
        - `<=`
          - 작거나 같냐

### 문자 (string)

- 사칙연산 비스무리한 연산들
    - `+`

      - 두 string을 붙인다. 영어로 concatenate 한다고 말한다.

      - 예)

        ```python
        >>> "a" + "B"
        'aB'
        ```

    - `*`
        
        - `문자 * 숫자`: 문자를 숫자만큼 반복한다 (여러번 concatenate 한다) 
        
        - 예)
        
          ```python
          >>> "wadu" * 2
          'waduwadu'
          ```
- 비교 연산
    - `==`
    - `!=`
    - `>`
    - `<`
    - `>=`
    - `<=`
- 길이
    - "a" -> 1
    - "ab" -> 2
    - len("asfdasdfas")
- membership
    - "a" in "ab"?
    - "a" not in "ab"?

- 참거짓
    - bool
        - not 
            - not True
            - not False
        - and
            - A and B 
                - True and True == True
                - True and False == False
                - False and True == False
                - False and False == False
            - 예쁘고 and 귀여운
            - 예뻐 안귀여워
            - 안예뻐 그리고 귀여워
            - 예뻐 그리고 귀여워
            - 안예뻐 그리고 안귀여워
        - or
            - A or B 
                - True or True == True
                - True or False == True
                - False or True == True
                - False or False == False
            - 예쁘거나 or 귀여우면
            - 예뻐 안귀여워
            - 안예뻐 그리고 귀여워
            - 예뻐 그리고 귀여워
            - 안예뻐 그리고 안귀여워

    - 예) 1210 이라는 숫자가
        - 2의 배수 이고 (and) 11의 배수이냐?
        - 1210이 2의 배수이냐?
            - 1210을 2로 나눈 나머지가 0인지?
            - 1210을 퍼센트
    - 예2) 60 이라는 숫자가
        - 2의 배수이거나 7의 배수이면 True를 보여줘 나한테
    - 예3) "bitcoin" 이라는 string에 "b"라는 letter가 있고 "c" 라는 letter가 있으면 True를 보여줘 나한테
